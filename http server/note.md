`跨域`指的是`JavaScript`通过`XMLHttpReques`t请求数据时，`调用JavaScript的页面`所在的域和`被请求页面`的域不一致。
对于网站来说，浏览器出于安全考虑是不允许跨域。
另外，对于`域`相同，但`端口或协议不同`时，浏览器也是`禁止`的。下表给出了进一步的说明：

|URL| 说明   | 是否允许请求|
| -| -----:|
| http://a.example.com/|
|http://a.example.com/a.txt     | $1600 |  
| 手机        |   \$12   |   
| 管线        |    \$1    |
